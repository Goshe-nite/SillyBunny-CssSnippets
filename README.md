# CSS Snippets

Adds a UI to manage custom CSS snippets.

Snippets can be globally activated, linked to a specific theme, or linked to a specific chat (character / group).

![](README/csss-01.png)

## Warning
This repo is forked from https://github.com/LenAnderson/SillyTavern-CssSnippets and originally from LenAnderson, thanks to his amazing work toggling and adding CSS snippets has never been easier.
This fork only serves as a band-aid fix to fix immediately problems when using the extension with https://github.com/platberlitz/SillyBunny, including removing the Files Plugin API import and it's related functions, and fixing the failed button append caused by change of html structure in SillyBunny.

## Installation

Use ST's inbuilt extension installer with this URL:  
https://github.com/Goshe-nite/SillyBunny-CssSnippets.git



## Usage

Open **User Settings** and click on the button next to **Custom CSS**
